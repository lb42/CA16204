Not pretty, but...

1. run grabGenerator to create script which downloads a bunch of Gutenberg titles.
2. do ls -1sS  > sizes.txt to get a list of their sizes and names 
3. run filter.xsl to extract bibls for all gutenberg texts from bassettPlus.xml  into gutenTitles.xml
4. run teitocsv.xsl on gutenTitles.xml  to produce gutenList.tmp
5. run addSizes.prl to merge size info from sizes.txt into new CSV version of gutenberg.csv

saxon gutenTitles.xml ../teitocsv.xsl >gutenList.tmp; perl addSizes.prl gutenList.tmp > gutenList.csv

6. Next step will be to merge the size info back into bassettPlus.xml
